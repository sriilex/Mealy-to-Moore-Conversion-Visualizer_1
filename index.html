🔵 Mealy Machine

A Mealy Machine is a finite state machine where the output depends on both the current state and the current input symbol.

M = (Q, Σ, Δ, δ, λ, q₀)
Q = finite set of states
Σ = input alphabet
Δ = output alphabet
δ: Q × Σ → Q    (transition function)
λ: Q × Σ → Δ    (output function — on transitions)
q₀ = initial state
  • Output is produced on each transition
  • For input of length n → produces n output symbols

🟠 Moore Machine

A Moore Machine is a finite state machine where the output depends only on the current state.

M = (Q, Σ, Δ, δ, λ, q₀)
Q = finite set of states
Σ = input alphabet
Δ = output alphabet
δ: Q × Σ → Q    (transition function)
λ: Q → Δ        (output function — on states)
q₀ = initial state
  • Output is associated with each state
  • For input of length n → produces n+1 output symbols

🔄 Mealy → Moore Conversion Algorithm

  1. Identify incoming outputs — For each Mealy state, find all outputs produced on transitions entering it.
  2. Create Moore states — For each (state, incoming-output) pair, create one Moore state [q, z].
  3. Handle start state — The start state gets [q₀, –] since no output has been produced yet.
  4. Build transitions — For Moore state [q, z] on input a: go to [δ(q,a), λ(q,a)] from the Mealy machine.
  5. Result — Both machines are equivalent and produce the same output for any input string.
